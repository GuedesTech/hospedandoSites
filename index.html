Passos para Hospdar um Projeto no GitHub Pages

  1. Crie um repositório no GitHub com o mesmo nome do seu projeto
  2. No terminal, inicalize o repositório local com git init
  3. Altere o nome do branch principal para main
  4. Adicione os arquivos com git add . o faça o commit com o git commit -m "fist commit"
  5. Conecte ao repositório remoto com:
    git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITÓRIO
  6. Envie os arquivos com git push -u origin main
  7. No GitHub, vá em Settings > Pages.
  8. Em "Branch", selecione main e a pasta / (rost). Clique em Save
  9. Após alguns segundos, seu site estará disponível no link exibido em "Pages"
